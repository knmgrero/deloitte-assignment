# Deloitte Assignment - Order API


## API Description
This API contains single resource "Order" and allows the following

#### /order
  ##### `GET`| List orders
  ##### `POST`| Create a new order 

#### /order/{id}
  ##### `GET`| Get order details by Id
  ##### `PUT`| Update an order by Id
  ##### `DELETE`| Delete an order by Id
